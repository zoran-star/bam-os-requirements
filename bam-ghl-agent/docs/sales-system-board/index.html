THE FUNNEL

Free-trial landing page

The academy's web page that turns a visitor into a lead. Brand, photos + selling points pulled from their settings.
THE LEAD FORM
  • Asks: parent name · phone · email · athlete name
  • Age + location - two qualification questions, asked up front
  • Submitting creates the contact record the agents work from
  • Core fields are locked (the agent depends on them) - you can add your own extra questions
THE BOOKING CALENDAR
  • Right after the form, they can pick a trial time on the spot
  • Book right there → they enter the pipeline already booked (straight to Confirm)
  • Skip it → the Booking agent picks them up
OTHER WAYS IN
  • Program pages (Elementary, High School)
  • Contact page only if this is the academy's only offer
TRACKED THE WHOLE WAY
  • Page views → forms filled → trials booked (feeds Metrics)
STAGE 1

Booking

A new lead just came in. One goal: get them booked into a free trial.
⚙ ENGINE: Booking Agent (AI) - texts the lead, you approve every message
ENTRY - how people arrive
  • The lead form on the landing page ←
  • No-shows + cancelled bookings coming back around
  • Anyone who replies from Ghosted or Nurture
WHAT THE AGENT KNOWS
  • Everything on the Agent Knowledge card ↓
  • Which form they came through + their first message
  • The qualification checklist - it asks what's missing
DATA SAVED HERE
  • New contact record: parent name · phone · email · athlete name + age · location answer · which page they came from
  • Every text, in + out, in one conversation thread
  • Qualification answers as they show up in the convo
  • Handoff notes for the other agents
EXIT - where they go next
  • Books a trial → Confirm
  • Goes quiet → Ghosted ↓
  • Not interested → Nurture ↓
STAGE 3

Confirm

The trial is booked. Make sure they actually show up.
⚙ ENGINE: Confirm Agent (AI) + instant confirmation text/email + 9am day-of check-in
ENTRY
  • Books a time on the trial calendar (from the agent or the landing page)
WHAT THE AGENT KNOWS
  • Everything on the Agent Knowledge card ↓
  • The exact booking: session, date, time, location
  • The full conversation so far + the Booking agent's notes
DATA SAVED HERE
  • The trial booking (session, date, time)
  • No-show / can't-make-it / cancelled status
  • A rebook note handed back to the Booking agent
EXIT
  • Showed up + good fit (trainer's post-trial form) → Closing
  • No-show / can't make it / cancels → back to Booking
  • No longer wants a trial → Nurture ↓
STAGE 4

Closing

They came and it went well. Help them take the next step: becoming a member.
⚙ ENGINE: Closing Agent (AI) - follows a 2-message follow-up plan
ENTRY
  • Good-fit trial done (from the post-trial form)
WHAT THE AGENT KNOWS
  • Everything on the Agent Knowledge card ↓
  • The trainer's post-trial notes ("thinking about it, deciding this weekend...")
  • Objection handling + sales strategies coming - Zoran + Mike session
DATA SAVED HERE
  • Post-trial form: showed up? good fit? which trainer + their notes
  • Follow-up progress (message 1 sent, message 2 sent)
  • The final mark: Won / Lost / Still deciding → feeds the close rate
EXIT
  • Enrolls → MEMBER →
  • Says no / ghosts the follow-ups → Nurture ↓
STAGE 2 - BELOW THE LINE

Ghosted

They stopped answering. Three quick texts to restart the conversation.
⏱ ENGINE: Automation - text on day 1, day 2, day 3 · sequence is yours to edit
ENTRY
  • Went quiet in Booking (24h+ of silence)
DATA SAVED HERE
  • Which day of the drip they're on · replies land in the same thread
EXIT
  • They reply → back to Booking
  • All 3 texts run dry → Nurture
STAGE 5 - THE LOST BENCH

Nurture

"Not now" doesn't mean never. Leads here are marked Lost - a slow drip keeps the door open.
⏱ ENGINE: Automation - emails at +1, +2, +3 weeks · sequence is yours to edit
ENTRY
  • Every "not interested", "undecided" or "said no" - from any stage
DATA SAVED HERE
  • Drip position · the reason they said no · the Lost mark
EXIT
  • They reply → back to Booking, and the Lost mark comes OFF
  • Whole drip runs dry → Unqualified

Member 🏆

They enrolled. Sales is done - the member-management system takes over from here (its own preset, confirmed later - onboarding data points live there).

Unqualified

The ONE dead end. Fully removed from the pipeline, never contacted again. The reason picked is saved on the contact.
QUALIFICATIONS - who's a fit
1 · Close enough to the academy
2 · Athlete age in program range
3 · Good fit for the program - judged at the trial
WHERE WE LEARN EACH ONE
  • The lead form asks age + location up front
  • The agents read them out of the conversation - and ask about whatever's missing
  • The trainer answers "good fit?" on the post-trial form
Marking someone Unqualified always means picking one of these (or "other"). Locked at three for every academy - your city + age range are the VALUES, set from your offer.

THE PACKAGE

Sales system preset

The whole playbook as one package: stages, engines, funnel, qualifications and metrics. The MACHINE is one master shared by every academy - edit it once, everyone updates. The messages inside it are seeded, then each academy's own.

Free trial is preset #1. More presets later (e.g. discovery-call-first) - same blocks, different arrangement.
THE FRONT DOOR

Funnel

The web path that turns a stranger into a lead: a landing page they visit, a form they fill, a calendar they book on. Every step is tracked so you see where people drop off.

Landing page → lead form → trial calendar. Program pages feed in too.
FUNNEL PIECE

Forms

The questions asked up front. Answers create the contact record and pre-fill the qualification checklist, so the agent doesn't have to ask twice. Core fields are locked; each academy can add its own extras.

Parent name · phone · athlete name · age + location (two qualification questions).
FUNNEL PIECE

Calendars

Bookable trial times. Booking a slot is what moves a lead forward - whether they book on the landing page or the agent books for them mid-conversation.

Book on the page → skip straight to Confirm. The 9am day-of check-in runs off this booking.
THE SPINE

Pipeline + stages

The journey map. Each stage is one job to get done with the lead - and a lead is always in exactly one stage, with exactly one engine working them.

Booking → Confirm → Closing on the main line, Ghosted + Nurture below it.
STAGE ANATOMY

Entry points

The ways someone arrives in a stage. Could be a form fill, a booking, a reply, or a hand-back from another stage.

Booking's entries: the lead form · no-shows returning · replies from Ghosted or Nurture.
STAGE ANATOMY

Engine

Who works the stage. Every stage has exactly one - an AI agent or an automation. No stage is ever unmanned.

Booking, Confirm + Closing run agents. Ghosted + Nurture run automations.
STAGE ANATOMY

Exit points

The ways out of a stage and where each one leads. Every hand-off between engines happens at an exit point - that's the only place it's defined.

Confirm's exits: showed up + good fit → Closing · no-show → Booking · no longer wants → Nurture.
ENGINE TYPE

Agents (AI)

An AI that reads the whole conversation and writes the next message itself, using the academy's real info. One voice and one craft for every academy - lessons taught anywhere sharpen everyone. It can move leads between stages and book trials. Nothing sends without approval.

Booking gets the trial booked · Confirm gets them to show up · Closing gets them enrolled.
ENGINE TYPE

Automations

Pre-written message sequences on a timer. We seed each academy's sequence tailored to them at onboarding - after that the copy, timing and steps are theirs to change. The moment a lead replies, the automation stops and an agent takes over.

Ghosted drip (day 1/2/3 texts) · Nurture drip (+1/+2/+3 week emails) · instant confirmations.
YOUR CONTROLS

Hawkeye

Mission control. Every agent suggestion lands here as a card you approve, change, or redirect. Your edits - with a quick "why" - become lessons the agents learn from.

Approve + send · change it · book them · mark lost or unqualified.
THE FILTER

Qualifications

The short list that defines who's a fit - locked at three for every academy (the city and age range are each academy's VALUES). Collected up front on the form, read out of conversations by the agents, judged at the trial by the trainer. Marking someone Unqualified always requires picking one.

Close enough · right age · good fit for the program.
THE BRAIN

Agent knowledge

Everything the agents know about the academy - pulled live from where each fact really lives (settings, the offer, staff records). Change it once, every agent updates. Never retyped.

Schedule, pricing, policies, coaches, selling points + the lessons you teach in Hawkeye.
THE SCORE

Metrics

The numbers the system calculates on its own, so you know if the machine is working - and where it leaks.

Qualified trial close rate is the health number. Plus trials booked, sales, funnel drop-off.
