COACHIQ

api-v3.coachiq.io

public API key (limited) + webhook
List products / users / bookings ❌ blocked
The public API key is auth-walled — admin/list queries return "must be logged in", and the dashboard host is WAF-blocked to server requests. No clean "list my products."
Create user ⚠️ via Zapier
signUp_V2 makes a bare account; the enrolling "Create User" runs through the Zapier integration.
Fire an automation (Add Product/Credits) ✅ webhook
POST /hook/automation/trigger/{id}
Bearer key · x-group-id · body {user:{id}}
Product ids ⚠️ via Stripe
Not from CoachIQ directly — harvested from the Stripe sub metadata.productId (above). New products with no sub yet = manual entry.
