STAGE 1

Booking

A new lead just came in. One goal: get them booked into a free trial.
⚙ ENGINE: Booking Agent (AI) - texts the lead, you approve every message
ENTRY - how people arrive
  • Free-trial form on the website (fed by the funnel ↓)
  • Contact form only if this is the academy's only offer
  • No-shows + cancelled bookings coming back around
  • Anyone who replies from Ghosted or Nurture
WHAT THE AGENT KNOWS
  • Everything on the Agent Knowledge card ↓
  • Which form they came through + their first message
  • The qualification checklist - it asks what's missing
DATA SAVED HERE
  • New contact record: parent name · phone · email · athlete name + age · location answer · which ad / page they came from
  • Every text, in + out, in one conversation thread
  • Qualification answers as they show up in the convo
  • Handoff notes for the other agents
EXIT - where they go next
  • Books a trial → Confirm
  • Goes quiet → Ghosted
  • Not interested → Nurture
STAGE 2

Ghosted

They stopped answering. Three quick texts to restart the conversation.
⏱ ENGINE: Automation - text on day 1, day 2, day 3
ENTRY
  • Went quiet in Booking (24h+ of silence)
DATA SAVED HERE
  • Which day of the drip they're on
  • Replies land in the same conversation thread
EXIT
  • They reply → back to Booking
  • All 3 texts run dry → Nurture
STAGE 3

Confirm

The trial is booked. Make sure they actually show up.
⚙ ENGINE: Confirm Agent (AI) + instant confirmation text/email + 9am day-of check-in
ENTRY
  • Books a time on the trial calendar
WHAT THE AGENT KNOWS
  • Everything on the Agent Knowledge card ↓
  • The exact booking: session, date, time, location
  • The full conversation so far + the Booking agent's notes
DATA SAVED HERE
  • The trial booking (session, date, time)
  • No-show / can't-make-it / cancelled status
  • A rebook note handed back to the Booking agent
EXIT
  • Showed up + good fit (trainer's post-trial form) → Closing
  • No-show / can't make it / cancels → back to Booking
  • No longer wants a trial → Nurture
STAGE 4

Closing

They came and it went well. Help them take the next step: becoming a member.
⚙ ENGINE: Closing Agent (AI) - follows a 2-message follow-up plan
ENTRY
  • Good-fit trial done (from the post-trial form)
WHAT THE AGENT KNOWS
  • Everything on the Agent Knowledge card ↓
  • The trainer's post-trial notes ("thinking about it, deciding this weekend...")
  • Objection handling + sales strategies coming - Zoran + Mike session
DATA SAVED HERE
  • Post-trial form: showed up? good fit? which trainer + their notes
  • Follow-up progress (message 1 sent, message 2 sent)
  • The final mark: Won / Lost / Still deciding → feeds the close rate
EXIT
  • Enrolls → MEMBER
  • Says no / ghosts the follow-ups → Nurture
STAGE 5

Nurture

"Not now" doesn't mean never. A slow drip keeps the door open.
⏱ ENGINE: Automation - emails at +1, +2, +3 weeks
ENTRY
  • Every "not interested", "undecided" or "said no" - from any stage
DATA SAVED HERE
  • Drip position + the reason they said no
EXIT
  • They reply → back to Booking
  • Whole drip runs dry → Unqualified

Human takeover

Complaints and anything weird skip the machine and land with a person in Hawkeye.

Member 🏆

They enrolled. Sales is done - the member-management system takes over from here (its own preset, confirmed later - onboarding data points live there).

Unqualified

The ONE dead end. Fully removed from the pipeline, never contacted again. The reason picked is saved on the contact.
