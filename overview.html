Requirements Whiteboard
whiteboard/
Tool ● Live

Standalone planning tool used to run structured review sessions for any project. Sessions are approved, rejected, or given feedback — then exported to Claude Code for execution. The tool itself lives here; session output files live inside each project's own folder.

React / Vite Notion API whiteboard-beta-indol.vercel.app
How it works
Review sessions in Notion
Sessions DB stores all cards. Each session has items to approve, reject, or give feedback on.
Export for AI
Click "Export for AI" → paste into Claude Code → Claude processes decisions, updates Notion + prototype.
