🏀 BAM GTA Phase 1 — First Live Deployment
🏀
Phase 1 Overview
bam-gta-phase1/
Phase 1

First live deployment of FullControl at BAM Basketball's GTA location. Three moving parts: staff dashboard, parent app, and the GHL automation layer connecting them. Real operators, real data, real proof.

Parent books class
→ GHL fires confirmation automation → Staff sees booking in Sessions
Trial ends
→ Staff marks outcome in Post-Trial → GHL fires conversion sequence
Payment fails
→ Stripe webhook → GHL dunning sequence → Staff sees in Failed Payments
Staff sends message
→ GHL routes to parent → Parent sees in Messages
